# Pink Restaurant

Developing a dinner preservation website in which clients
are allowed to pre-order at the desired restaurant in pre-defined time
slots. Besides, the system also updates the restaurants status for ad-
mins. 
---
#Tech Stack: 
- Java Sevlet
- JSP
- JavaScript
- HTML/CSS
